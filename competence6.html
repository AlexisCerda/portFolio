Compétence 6 : S’adapter aux évolutions du numérique

Quel était le projet ?

Conception et développement d’un site web présentant l’ensemble des SAÉ réalisées pendant le semestre, dans un format de portfolio interactif.

Organisation collaborative

Travail en binôme avec mise en commun du contenu. L’un se chargeait du design, l’autre de l’intégration et de l’adaptation responsive.

Étapes, difficultés

  • Structuration des pages HTML/CSS
  • Utilisation d’animations et de médias
  • Responsive design pour affichage mobile

Problèmes de compatibilité résolus grâce aux outils de debug navigateur.

Traces justificatives

  • Capture du site portfolio en ligne
  • Code source déposé dans le dépôt Git

Point d'amélioration

Intégrer une base de données ou un CMS pour automatiser la gestion du contenu.
